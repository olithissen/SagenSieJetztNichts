# SagenSieJetztNichts
Eine Chrome-Extension, die die Kommentarbereiche der beliebtesten deutschen Nachrichtenseiten entfernt.

# Installation
SagenSieJetztNichts.crx herunterladen und per Drag&Drop in den Chrome-Browser ziehen.
